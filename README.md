# Web Scraper (clutch.co)
## Run Instructions
### Linux
```
git clone https://github.com/gandhisamay/web_scraper.git
cd web_scraper
bash run.sh
```

### Windows
Install the libraries given in the requirements.txt file
```
git clone https://github.com/gandhisamay/web_scraper.git
cd web_scraper
python3 setup.py
python3 company_scraper.py
python3 founder_scraper.py
```
